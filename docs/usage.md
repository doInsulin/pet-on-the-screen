# 小不点桌宠操作文档

## 启动

打开 PowerShell：

```powershell
cd D:\forFun\bichon-desktop-pet
npm.cmd run dev
```

启动后桌面上会出现一只透明背景的小狗。PowerShell 里建议用 `npm.cmd`，不要直接用 `npm`，否则可能被执行策略拦住。

## 关闭

把鼠标移到小狗身上，右键单击即可退出。

也可以在 Windows 托盘里找到 `Bichon Desktop Pet`，右键选择 `Quit`。

## 移动位置

按住小狗所在的小窗口拖动即可移动。

当前窗口没有边框、气泡和按钮，只保留小狗本体。

## 互动方式

单击小狗：摸头，小狗会进入开心/被摸状态。

双击小狗：触发拔河动作。这个动作模拟小不点咬住屏幕外的人手/玩具，用力往后一拽一拽，并盯着鼠标方向看。

右键小狗：退出程序。

窗口聚焦时，可以用下面的测试快捷键直接触发每一种动效：

| 快捷键 | 动效 | 说明 |
| --- | --- | --- |
| `I` | 发呆 idle | 回到安静陪伴状态 |
| `H` | 摸头 pat | 和单击小狗相同 |
| `C` | 贴贴 cuddle | 粘人贴贴状态 |
| `P` | 跑酷 parkour | 小不点在屏幕上跑来跑去，结束后回到右下角 |
| `T` | 拔河 tug | 和双击小狗相同 |
| `F` | 讨食 beg | 摇尾巴、找你要吃的 |
| `M` | 吃肉/食物 food | 听到吃饭或肉肉后的兴奋状态 |
| `W` | 喝水 drink | 喝水休息 |
| `D` | 刨窝 dig | 睡前刨窝/整理位置 |
| `L` | 趴下 flop | 趴下放松 |
| `S` | 睡觉 sleep | 进入长时间睡觉状态 |
| `N` | 闻闻 sniff | 闻闻找吃的/探索 |

不操作时，小狗会根据精力、困意、饥饿、粘人程度、玩心和好奇心自己切换状态，包括：

- 发呆陪你
- 粘人贴贴
- 跑酷：会推动整个桌宠窗口在屏幕上跑来跑去
- 拔河：不显示中间绳子，表现为和屏幕外的人互动
- 听到吃饭/肉肉飞奔
- 喝水：跑酷结束后会去喝水
- 刨窝：长时间睡觉前会先刨窝
- 趴下/睡觉：包含蜷缩成一团、仰着睡、趴着睡等形态；没人陪一段时间会睡很久
- 闻闻找吃的

## 小不点设定

当前设定文件：

```text
src/pet-profile.json
```

里面记录了小不点的名字、别名、颜色、玩具和行为权重。现在的设定包括：

- 名字：小不点
- 别名：不点点、点小姐
- 白色小体比熊
- 一岁半
- 没有项圈
- 玩具：蓝色小飞象、胡萝卜玩偶、拔河绳
- 性格：活泼、调皮、粘人、爱玩、听到吃饭会冲过来

## 调整性格和行为频率

打开：

```text
src/pet-profile.json
```

修改 `behaviorWeights`。数字越大，越容易出现这个行为。

例如想让小不点更常睡觉：

```json
"sleep": 24
```

想让小不点更常跑酷：

```json
"parkour": 35
```

保存后重启：

```powershell
npm.cmd run dev
```

## 当前限制

当前默认使用 `参考/gpt_generate比熊.png` 这张 4x4 参考图作为小不点的 sprite sheet，并保留代码绘制版本作为兜底。由于参考图不是严格透明分格的专业 sprite sheet，个别动作仍可能受原图构图影响；后续如果有逐帧透明 PNG 或更规范的 sprite sheet，可以继续替换当前素材。
